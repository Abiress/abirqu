## 2. Reporting a Vulnerability
Vulnerabilities can be reported via:
- Private GitHub issue on the AbirQu repository
- Direct contact with Abir Maheshwari via [GitHub profile](https://github.com/abirmaheshwari)

Response timeline:
- Acknowledgment within 72 hours
- Initial assessment within 7 days
- Coordinated disclosure after fix deployment

## 3. Built-in Security Features
- **Circuit Encryption**: AES-256 encryption for stored/transmitted circuit payloads
- **QKD Simulation**: BB84 protocol simulation with eavesdropping detection
- **Hardware Attestation**: Verify backend integrity before execution
- **Circuit Obfuscation**: Prevent reverse engineering of proprietary circuits
- **No Telemetry**: No user data collection without explicit consent
