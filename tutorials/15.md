## Step-by-Step Code

```python
import math
from abirqu import Circuit
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Quantum counting
# ============================================

print("Quantum Counting Algorithm:")
print("=" * 50)

def quantum_counting_circuit(num_qubits, num_solutions):
    """Build quantum counting circuit.
    
    Estimates number of solutions M in search space N.
    """
    num_counting = 3  # Counting qubits
    total = num_counting + num_qubits
    circuit = Circuit(total, name="QCount")
    
    # Initialize counting qubits in superposition
    for i in range(num_counting):
        circuit.h(i)
    
    # Initialize search qubits in superposition
    for i in range(num_counting, total):
        circuit.h(i)
    
    # Apply controlled Grover operator
    for i in range(num_counting):
        # Simplified Grover iteration
        for j in range(num_counting, total):
            circuit.h(j)
            circuit.x(j)
        circuit.z(total - 1)
        for j in range(num_counting, total):
            circuit.x(j)
            circuit.h(j)
    
    # Inverse QFT on counting qubits
    for i in range(num_counting):
        circuit.h(i)
    
    # Measure counting qubits
    circuit.measure_all()
    
    return circuit

# Test counting
N = 8  # Search space size
M_true = 2  # Number of solutions

circuit = quantum_counting_circuit(num_qubits=3, num_solutions=M_true)
result = QuantumRun(circuit, shots=1000)

# Convert to solution count
counts = result.counts
most_common = max(counts, key=counts.get)
theta = int(most_common, 2) * math.pi / (2 ** 3)
M_estimated = N * math.sin(theta) ** 2

print(f"Search space N = {N}")
print(f"True solutions M = {M_true}")
print(f"Estimated solutions M ≈ {M_estimated:.1f}")
print(f"Measurement counts: {counts}")

# ============================================
# Part 2: Scaling analysis
# ============================================

print("\n\nScaling Analysis:")
print("=" * 50)

print("N (search space) | M (solutions) | Classical queries | Quantum queries")
print("-" * 70)

for N_log in [3, 4, 5, 6, 7, 8]:
    N = 2 ** N_log
    M = max(1, N // 8)  # 1/8 of solutions
    classical = N // M
    quantum = int(math.pi / 4 * math.sqrt(N / M))
    print(f"{N:>16} | {M:>13} | {classical:>16} | {quantum:>14}")
```

## Key Concepts

### How Quantum Counting Works
1. Combine phase estimation with Grover operator
2. Phase gives information about eigenvalues
3. Eigenvalues relate to number of solutions

### Formula
- Estimated solutions: M = N × sin²(πθ)
- Where θ is the measured phase

### Advantages
- Counts solutions without finding them
- Quadratic speedup over classical counting
- Useful for database analysis

### Applications
- **Database analysis**: Count records matching criteria
- **Optimization**: Estimate number of optimal solutions
- **Statistical sampling**: Estimate probabilities
