## Step-by-Step Code

```python
import math
from abirqu import Circuit, H, X, CNOT, Measure
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Quantum walk on a line (4 nodes)
# ============================================

print("Quantum Walk on 4-node line:")
print("=" * 50)

def build_quantum_walk(num_nodes, steps):
    """Build quantum walk circuit.
    
    Uses 2 qubits for position (4 nodes) and 1 qubit for coin.
    """
    circuit = Circuit(3, name="QWalk")
    
    # Initialize at position |00⟩ with coin |0⟩
    # Apply Hadamard to coin qubit
    circuit.h(2)  # Coin qubit
    
    # Walk steps
    for step in range(steps):
        # Coin flip (Hadamard on coin)
        circuit.h(2)
        
        # Conditional shift based on coin
        # If coin=1, shift right
        circuit.cnot(2, 0)  # Shift based on coin
        circuit.cnot(2, 1)
    
    return circuit

# Run walk for different step counts
for steps in [1, 2, 3]:
    walk = build_quantum_walk(num_nodes=4, steps=steps)
    walk.measure([0, 1, 2])
    
    result = QuantumRun(walk, shots=1000)
    
    # Separate position and coin measurements
    position_counts = {}
    for bitstring, count in result.counts.items():
        pos = bitstring[:2]  # First 2 bits = position
        position_counts[pos] = position_counts.get(pos, 0) + count
    
    print(f"Steps: {steps} | Position distribution: {position_counts}")

# ============================================
# Part 2: Quantum walk on cycle (4 nodes)
# ============================================

print("\n\nQuantum Walk on 4-node cycle:")
print("=" * 50)

def build_cycle_walk(num_nodes, steps):
    """Build quantum walk on a cycle graph."""
    circuit = Circuit(3, name="CycleWalk")
    
    # Coin qubit in superposition
    circuit.h(2)
    
    for step in range(steps):
        # Coin flip
        circuit.h(2)
        
        # Shift on cycle (modular addition)
        # Simplified: XOR with coin for modular behavior
        circuit.cnot(2, 0)
        circuit.cnot(2, 1)
        
        # Wrap around (cycle property)
        circuit.x(2)
    
    return circuit

cycle_walk = build_cycle_walk(num_nodes=4, steps=2)
cycle_walk.measure([0, 1, 2])

result = QuantumRun(cycle_walk, shots=1000)
print(f"Cycle walk results: {result.counts}")

# ============================================
# Part 3: Quantum vs Classical spreading
# ============================================

print("\n\nSpreading speed comparison:")
print("=" * 50)
print("Classical walk: spreads as O(√t)")
print("Quantum walk:   spreads as O(t)")
print("Quadratic speedup in search and exploration!")
```

## Key Concepts

### Quantum Walk Components
1. **Coin space**: Determines direction (like flipping a coin)
2. **Position space**: Where the walker is on the graph
3. **Coin operator**: H or other unitary on coin qubit
4. **Shift operator**: Moves walker based on coin state

### Quantum vs Classical
| Property | Classical Walk | Quantum Walk |
|----------|---------------|--------------|
| Spreading | O(√t) | O(t) |
| Distribution | Binomial | Interference pattern |
| Applications | Random algorithms | Search, simulation |

### Applications
- **Graph search**: Finding marked nodes faster
- **Quantum simulation**: Simulating physical systems
- **Machine learning**: Graph neural networks
- **Optimization**: Combinatorial problems
