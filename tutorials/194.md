# Tutorial 194: Quantum Feature Engineering

## Objective
Create and optimize quantum features for machine learning models using quantum circuits and encoding strategies.

## Prerequisites
- Tutorials 1-193 completed
- AbirQu installed (`pip install abirqu`)

## What You'll Learn
- Quantum feature maps
- Feature selection and optimization
- Quantum-classical feature fusion

## Step-by-Step Code

```python
import math
import numpy as np
from abirqu import Circuit
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Quantum Feature Maps
# ============================================

print("Quantum Feature Engineering:")
print("=" * 50)

print("Feature Encoding Strategies:")
encoding_strategies = [
    ("Amplitude Encoding", "Encode in qubit amplitudes", "log(n) qubits"),
    ("Angle Encoding", "Encode in rotation angles", "n qubits"),
    ("Basis Encoding", "Encode in computational basis", "n qubits"),
    ("IQP Encoding", "Instantaneous Quantum Polynomial", "n qubits"),
    ("ZZ Feature Map", "ZZ entangling feature map", "n qubits"),
]

for strategy, description, qubits_needed in encoding_strategies:
    print(f"  {strategy:20s}: {description:35s} | Qubits: {qubits_needed}")

# Demonstrate angle encoding
circuit = Circuit(4, name="AngleEncoding")
features = [0.5, 1.2, 0.8, 1.5]

for i, feature in enumerate(features):
    circuit.ry(i, feature)

circuit.measure_all()

result = QuantumRun(circuit, shots=500)
print(f"\nAngle encoding result: {dict(list(result.counts.items())[:4])}")

# ============================================
# Part 2: Feature Selection and Optimization
# ============================================

print("\n\nFeature Selection and Optimization:")
print("-" * 50)

feature_set = [
    ("Feature-1", 0.85, "High"),
    ("Feature-2", 0.72, "Medium"),
    ("Feature-3", 0.91, "High"),
    ("Feature-4", 0.45, "Low"),
    ("Feature-5", 0.78, "Medium"),
    ("Feature-6", 0.88, "High"),
]

print("Feature Importance Scores:")
for feature, score, importance in feature_set:
    print(f"  {feature:12s}: {score:.2f} ({importance})")

optimization_techniques = [
    ("Principal Component Analysis", "Reduce dimensionality"),
    ("Quantum Feature Selection", "Select most informative features"),
    ("Mutual Information", "Measure feature-target dependency"),
    ("Recursive Feature Elimination", "Iteratively remove features"),
]

print("\nOptimization Techniques:")
for technique, description in optimization_techniques:
    print(f"  {technique:35s}: {description}")

circuit = Circuit(3, name="FeatureOptimized")
circuit.ry(0, 0.85)
circuit.ry(1, 0.91)
circuit.ry(2, 0.88)
circuit.cnot(0, 1)
circuit.cnot(1, 2)
circuit.measure_all()

result = QuantumRun(circuit, shots=300)
print(f"\nOptimized features: {result.counts}")

# ============================================
# Part 3: Quantum-Classical Feature Fusion
# ============================================

print("\n\nQuantum-Classical Feature Fusion:")
print("-" * 50)

fusion_methods = [
    ("Concatenation", "Append quantum features to classical"),
    ("Attention", "Weighted combination of feature types"),
    ("Cross-attention", "Interact quantum and classical features"),
    ("Ensemble", "Separate models with combined predictions"),
]

for method, description in fusion_methods:
    print(f"  {method:20s}: {description}")

# Demonstrate fusion
classical_features = np.array([0.1, 0.2, 0.3, 0.4])
quantum_features = np.array([0.5, 0.6, 0.7, 0.8])

fused_features = np.concatenate([classical_features, quantum_features])
print(f"\nClassical features: {classical_features}")
print(f"Quantum features: {quantum_features}")
print(f"Fused features: {fused_features}")

circuit = Circuit(4, name="FeatureFusion")
for i, feature in enumerate(quantum_features):
    circuit.ry(i, feature)
circuit.h(0)
circuit.cnot(0, 1)
circuit.cnot(2, 3)
circuit.measure_all()

result = QuantumRun(circuit, shots=400)
print(f"\nFusion result: {result.counts}")

print("\n" + "=" * 50)
print("Quantum Feature Engineering Complete!")
print("=" * 50)
```

## Key Concepts

### Quantum Feature Maps
- Encode classical data into quantum states
- Different encoding strategies for different data types
- Efficient qubit usage with amplitude encoding

### Feature Optimization
- Select most informative features for quantum models
- Reduce dimensionality while preserving information
- Validate feature quality with quantum metrics

### Quantum-Classical Fusion
- Combine quantum and classical feature representations
- Leverage strengths of both feature types
- Ensemble methods for robust predictions
