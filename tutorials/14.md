## Step-by-Step Code

```python
import math
from abirqu import Circuit, H, X, CNOT, Rz, Measure
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Simple amplitude estimation
# ============================================

print("Quantum Amplitude Estimation:")
print("=" * 50)

def build_amplitude_estimation(target_amplitude, num_counting=3):
    """Build amplitude estimation circuit.
    
    Estimates amplitude a where P(|1⟩) = a²
    """
    total_qubits = num_counting + 1
    circuit = Circuit(total_qubits, name="AE")
    
    # Initialize counting qubits in superposition
    for i in range(num_counting):
        circuit.h(i)
    
    # Prepare target state with amplitude a
    # Ry(2*arcsin(a)) on target qubit
    angle = 2 * math.asin(target_amplitude)
    circuit.ry(num_counting, angle)
    
    # Controlled Grover iterations
    for i in range(num_counting):
        # Controlled Oracle (simplified)
        circuit.cnot(i, num_counting)
    
    # Inverse QFT on counting qubits
    for i in range(num_counting):
        circuit.h(i)
    
    # Measure counting qubits
    circuit.measure(list(range(num_counting)))
    
    return circuit

# Test with different amplitudes
amplitudes = [0.5, 0.7, 0.9]

for a_true in amplitudes:
    circuit = build_amplitude_estimation(a_true, num_counting=3)
    result = QuantumRun(circuit, shots=1000)
    
    # Convert measurement to amplitude estimate
    counts = result.counts
    most_common = max(counts, key=counts.get)
    phase = int(most_common, 2) / (2 ** 3)
    a_estimated = math.sin(phase * math.pi)
    
    print(f"True amplitude: {a_true:.2f} | "
          f"Estimated: {a_estimated:.2f} | "
          f"Error: {abs(a_estimated - a_true):.4f}")

# ============================================
# Part 2: Confidence intervals
# ============================================

print("\n\nConfidence Intervals:")
print("=" * 50)

circuit = build_amplitude_estimation(0.7, num_counting=3)
result = QuantumRun(circuit, shots=1000)

print(f"Measurement counts: {result.counts}")
print(f"Total shots: {sum(result.counts.values())}")
print(f"Confidence increases with more counting qubits and shots")
```

## Key Concepts

### How Amplitude Estimation Works
1. Prepare state with unknown amplitude a
2. Apply controlled Grover operator
3. Use QFT to extract phase
4. Phase relates to amplitude: a = sin(πθ)

### Quadratic Speedup
- Classical Monte Carlo: O(1/ε²) for error ε
- Quantum AE: O(1/ε) — quadratic speedup!

### Applications
- **Finance**: Option pricing, risk analysis
- **Chemistry**: Integral estimation
- **Machine learning**: Kernel estimation
- **Optimization**: Objective function estimation
