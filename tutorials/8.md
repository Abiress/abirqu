## Step-by-Step Code

```python
from abirqu import Circuit, H, X, Y, Z
from abirqu.primitives import QuantumRun
from abirqu.library import grover_circuit, grover_oracle

# ============================================
# Part 1: Using built-in Grover
# ============================================

print("Grover's Search (built-in):")
print("=" * 40)

# Search for |11⟩ in a 2-qubit space
grover = grover_circuit(num_qubits=2, target=3)
grover.measure_all()

result = QuantumRun(grover, shots=1000)
print(f"Target: |11⟩")
print(f"Counts: {result.counts}")
print(f"Most likely: {max(result.counts, key=result.counts.get)}")

# ============================================
# Part 2: Manual Grover implementation
# ============================================

print("\n\nManual Grover (3 qubits, target=|101⟩):")
print("=" * 40)

def build_grover(num_qubits, target, iterations):
    """Build Grover's search circuit."""
    circuit = Circuit(num_qubits, name="Grover")
    
    # Initialize superposition
    for i in range(num_qubits):
        circuit.h(i)
    
    # Grover iterations
    for _ in range(iterations):
        # Oracle: flip phase of target state
        oracle = grover_oracle(num_qubits, target)
        for gate in oracle.gates:
            circuit.add_gate(gate.name, gate.qubits, gate.params)
        
        # Diffusion operator
        for i in range(num_qubits):
            circuit.h(i)
            circuit.x(i)
        
        # Multi-controlled Z
        circuit.z(num_qubits - 1)
        
        for i in range(num_qubits):
            circuit.x(i)
            circuit.h(i)
    
    return circuit

# Optimal iterations for 3 qubits: π/4 * √(2³) ≈ 2.22 → 2 iterations
grover3 = build_grover(num_qubits=3, target=5, iterations=2)
grover3.measure_all()

result = QuantumRun(grover3, shots=1000)
print(f"Target: |101⟩")
print(f"Counts: {result.counts}")
print(f"Most likely: {max(result.counts, key=result.counts.get)}")

# ============================================
# Part 3: Scaling Grover
# ============================================

print("\n\nGrover scaling (search for |1111⟩ in 4-qubit space):")
print("=" * 40)

grover4 = grover_circuit(num_qubits=4, target=15)
grover4.measure_all()

result = QuantumRun(grover4, shots=1000)
print(f"Target: |1111⟩")
print(f"Counts: {result.counts}")
top = sorted(result.counts.items(), key=lambda x: -x[1])[:3]
print(f"Top 3: {top}")
```

## Key Concepts

### Grover's Algorithm Steps
1. **Initialize**: Apply H to all qubits (equal superposition)
2. **Oracle**: Mark the target state by flipping its phase
3. **Diffusion**: Amplify the marked state's amplitude
4. **Repeat**: Steps 2-3 for optimal number of iterations

### Optimal Iterations
For N items and M solutions:
- Iterations ≈ (π/4) × √(N/M)
- For 2³=8 items, 1 solution: ≈ 2 iterations
- For 2⁴=16 items, 1 solution: ≈ 3 iterations

### Speedup
- Classical: O(N) queries
- Grover: O(√N) queries
- Quadratic speedup

### Oracle Construction
The oracle flips the phase of the target state |101⟩:
- Apply X to qubits that should be 0
- Apply multi-controlled Z
- Apply X again (undo the flips)
