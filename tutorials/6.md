```python
import math
from abirqu import Circuit, H, Rz, SWAP, Measure
from abirqu.primitives import QuantumRun
from abirqu.library import qft_circuit

# ============================================
# Part 1: Using the built-in QFT
# ============================================

print("QFT using built-in function:")
print("=" * 40)

qft = qft_circuit(num_qubits=3)
from abirqu.visualization import draw
print(draw(qft, output="text"))

# ============================================
# Part 2: Manual QFT construction
# ============================================

print("\nManual QFT construction (3 qubits):")
print("=" * 40)

def build_qft(n):
    """Build QFT circuit for n qubits."""
    circuit = Circuit(n, name=f"QFT-{n}")
    
    for i in range(n):
        # Hadamard on qubit i
        circuit.h(i)
        
        # Controlled rotations
        for j in range(i + 1, n):
            angle = math.pi / (2 ** (j - i))
            circuit.rz(j, angle)  # Simplified controlled-Rz
    
    # Swap qubits for proper ordering
    for i in range(n // 2):
        circuit.swap(i, n - 1 - i)
    
    return circuit

manual_qft = build_qft(3)
print(draw(manual_qft, output="text"))

# ============================================
# Part 3: QFT on a specific state
# ============================================

print("\nQFT on |101⟩ (decimal 5):")
print("=" * 40)

# Prepare |101⟩
prep = Circuit(3)
prep.x(0)  # |001⟩
prep.x(2)  # |101⟩

# Apply QFT
qft_circ = qft_circuit(3)

# Combine circuits
full = Circuit(3)
# Add prep gates
for gate in prep.gates:
    full.add_gate(gate['name'], gate['qubits'])
# Add QFT gates
for gate in qft_circ.gates:
    full.add_gate(gate['name'], gate['qubits'])

full.measure([0, 1, 2])
result = QuantumRun(full, shots=1000)
print(f"QFT|101⟩ counts: {result.counts}")
```

## Key Concepts

### What QFT Does
The QFT transforms from the computational basis to the frequency basis:
- Input: |j⟩ (computational basis)
- Output: (1/√N) Σₖ e^(2πijk/N) |k⟩ (frequency basis)

### Applications
1. **Shor's Algorithm** — factoring large numbers
2. **Phase Estimation** — finding eigenvalues
3. **Quantum Signal Processing** — signal analysis
4. **Hidden Subgroup Problem** — period finding

### Complexity
- Classical FFT: O(N log N)
- QFT: O(n²) gates where n = log₂(N)
- Exponential speedup for quantum computers
