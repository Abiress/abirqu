```python
import math

# ============================================
# Part 1: Quantum complexity classes
# ============================================

print("Quantum Complexity (Expert II):")
print("=" * 50)

classes = {
    'BQP': {
        'definition': 'Bounded-error Quantum Polynomial time',
        'contains': 'P',
        'contained_in': 'PSPACE',
        'examples': ['Factoring', 'Simulation'],
    },
    'QMA': {
        'definition': 'Quantum Merlin-Arthur',
        'contains': 'NP',
        'contained_in': 'PP',
        'examples': ['Local Hamiltonian'],
    },
    'QCMA': {
        'definition': 'Quantum Classical Merlin-Arthur',
        'contains': 'NP',
        'contained_in': 'QMA',
        'examples': ['Quantum circuit evaluation'],
    },
    'QIP': {
        'definition': 'Quantum Interactive Proofs',
        'contains': 'IP',
        'contained_in': 'PSPACE',
        'examples': ['Quantum zero-knowledge'],
    },
    'AWPP': {
        'definition': 'Almost Wide Polynomial Page',
        'contains': 'BQP',
        'contained_in': 'PP',
        'examples': ['Approximation counting'],
    },
}

for name, properties in classes.items():
    print(f"\n{name}:")
    for prop, value in properties.items():
        print(f"  {prop}: {value}")

# ============================================
# Part 2: Lower bounds
# ============================================

print("\n\nQuantum Lower Bounds:")
print("-" * 50)

lower_bounds = [
    ("Search", "Ω(√N)", "Grover's optimal"),
    ("Element Distinctness", "Ω(N^{2/3})", "Ambainis"),
    ("Parity", "Ω(N)", "No speedup"),
    ("OR", "Ω(√N)", "Grover's optimal"),
    ("AND-OR Tree", "Ω(N^{5/7})", "Optimal"),
    ("Graph Connectivity", "Ω(N)", "No speedup"),
]

for problem, bound, note in lower_bounds:
    print(f"  {problem:25s}: {bound:10s} ({note})")

# ============================================
# Part 3: Quantum advantage
# ============================================

print("\n\nQuantum Advantage:")
print("-" * 50)

advantages = [
    ("Factoring", "Exponential", "Shor's algorithm"),
    ("Simulation", "Exponential", "Hamiltonian simulation"),
    ("Search", "Quadratic", "Grover's algorithm"),
    ("Optimization", "Unknown", "QAOA, annealing"),
    ("Machine Learning", "Polynomial-Exponential", "QSVM, QPCA"),
    ("Linear Systems", "Exponential", "HHL algorithm"),
    ("Counting", "Quadratic", "Quantum counting"),
]

for problem, speedup, algorithm in advantages:
    print(f"  {problem:20s}: {speedup:15s} ({algorithm})")

print("\nOpen Problems:")
print("  - Is BQP = BPP?")
print("  - Is factoring in BPP?")
print("  - What is the power of quantum proofs?")
print("  - Is QMA = PP?")
print("  - Is there quantum advantage for optimization?")
```

## Key Concepts

### Complexity Classes
- **BQP**: Efficiently solvable quantumly
- **QMA**: Quantum proofs help
- **QCMA**: Classical proofs help
- **QIP**: Interactive proofs

### Lower Bounds
- Prove algorithms are optimal
- No faster algorithm exists
- Fundamental limits

### Quantum Advantage
- **Exponential**: Shor, simulation, HHL
- **Quadratic**: Grover, search, counting
- **Unknown**: Many problems

### Open Problems
- **BQP vs BPP**: Do quantum computers solve more?
- **Factoring**: Is it in BPP?
- **QMA vs NP**: Power of quantum proofs
