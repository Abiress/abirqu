## Step-by-Step Code

```python
import math
import numpy as np
from abirqu import Circuit
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Simple anomaly detection
# ============================================

print("Quantum Anomaly Detection:")
print("=" * 50)

def quantum_anomaly_circuit(data_point, params):
    """Build circuit for anomaly detection."""
    circuit = Circuit(2, name="AnomalyDet")
    
    # Encode data point
    circuit.ry(0, data_point[0] * math.pi)
    circuit.ry(1, data_point[1] * math.pi)
    
    # Parameterized anomaly scoring
    circuit.ry(0, params[0])
    circuit.cnot(0, 1)
    circuit.ry(1, params[1])
    
    circuit.measure_all()
    return circuit

def compute_anomaly_score(data_point, params):
    """Compute anomaly score for a data point."""
    circuit = quantum_anomaly_circuit(data_point, params)
    result = QuantumRun(circuit, shots=100)
    
    # Score based on measurement distribution
    counts = result.counts
    p00 = counts.get('00', 0) / 100
    p11 = counts.get('11', 0) / 100
    
    # High score if measurement is unusual
    return abs(p00 - p11)

# ============================================
# Part 2: Training
# ============================================

print("\nTraining:")
print("-" * 50)

# Normal data (clustered)
normal_data = [[0.5, 0.5], [0.6, 0.4], [0.4, 0.6], [0.5, 0.5]]

# Anomaly data (far from cluster)
anomaly_data = [[0.1, 0.9], [0.9, 0.1], [0.0, 1.0]]

# Train parameters to minimize score on normal data
params = [0.0, 0.0]

for epoch in range(20):
    total_score = 0
    for point in normal_data:
        score = compute_anomaly_score(point, params)
        total_score += score
        
        # Update parameters to minimize score
        params[0] -= 0.05 * score
        params[1] -= 0.05 * score
    
    avg_score = total_score / len(normal_data)
    if (epoch + 1) % 5 == 0:
        print(f"  Epoch {epoch+1}: Avg normal score = {avg_score:.4f}")

# ============================================
# Part 3: Detection
# ============================================

print("\n\nAnomaly Detection Results:")
print("=" * 50)

print("Normal data points:")
for point in normal_data:
    score = compute_anomaly_score(point, params)
    print(f"  {point}: Score = {score:.4f} {'✓ Normal' if score < 0.3 else '✗ Anomaly'}")

print("\nAnomaly data points:")
for point in anomaly_data:
    score = compute_anomaly_score(point, params)
    print(f"  {point}: Score = {score:.4f} {'✓ Normal' if score < 0.3 else '✗ Anomaly'}")
```

## Key Concepts

### Quantum Anomaly Detection
1. **Encode** data points as quantum states
2. **Learn** normal distribution using parameterized circuit
3. **Score** new points based on how well they fit
4. **Threshold** scores to detect anomalies

### Quantum Advantages
- Can learn complex distributions
- Natural for quantum data
- Potential speedup in high dimensions

### Applications
- **Fraud detection**: Financial anomalies
- **Network security**: Intrusion detection
- **Medical diagnosis**: Disease detection
- **Manufacturing**: Defect detection
