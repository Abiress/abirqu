## Step-by-Step Code

```python
import math
from abirqu import Circuit, H, X, Y, Z
from abirqu.primitives import QuantumRun
from abirqu.library import qaoa_circuit, qaoa_maxcut

# ============================================
# Part 1: Using built-in QAOA
# ============================================

print("QAOA for Max-Cut (built-in):")
print("=" * 50)

# Simple graph: 3 nodes, edges: (0,1), (1,2), (0,2)
edges = [(0, 1), (1, 2), (0, 2)]
qaoa = qaoa_maxcut(num_qubits=4, edges=edges, p=1)

from abirqu.visualization import draw
print(draw(qaoa, output="text"))

# ============================================
# Part 2: Manual QAOA implementation
# ============================================

print("\nManual QAOA (3 qubits, p=2):")
print("=" * 50)

def build_qaoa(edges, num_qubits, p):
    """Build QAOA circuit."""
    circuit = Circuit(num_qubits, name="QAOA")
    
    # Initial superposition
    for i in range(num_qubits):
        circuit.h(i)
    
    # QAOA layers
    for layer in range(p):
        # Problem Hamiltonian (ZZ interactions)
        for i, j in edges:
            circuit.cnot(i, j)
            circuit.rz(j, math.pi / 2)  # γ parameter
            circuit.cnot(i, j)
        
        # Mixer Hamiltonian (X rotations)
        for i in range(num_qubits):
            circuit.rx(i, math.pi / 2)  # β parameter
    
    return circuit

qaoa_manual = build_qaoa(edges, num_qubits=3, p=2)
qaoa_manual.measure_all()

result = QuantumRun(qaoa_manual, shots=1000)
print(f"Measurement counts: {result.counts}")

# Find best cut
print("\nMax-Cut solutions:")
for bitstring, count in sorted(result.counts.items(), key=lambda x: -x[1])[:3]:
    # Calculate cut value
    cut = 0
    for i, j in edges:
        if bitstring[i] != bitstring[j]:
            cut += 1
    print(f"  |{bitstring}⟩: cut={cut}, count={count}")

# ============================================
# Part 3: Optimization
# ============================================

print("\n\nQAOA Optimization (simulated):")
print("=" * 50)

# In real QAOA, we'd optimize γ and β parameters
# Here we show the concept
best_cut = 0
best_config = None

for config in ['000', '001', '010', '011', '100', '101', '110', '111']:
    cut = sum(1 for i, j in edges if config[i] != config[j])
    if cut > best_cut:
        best_cut = cut
        best_config = config

print(f"Optimal: |{best_config}⟩ with cut = {best_cut}")
print(f"Max possible cut for triangle graph = 2")
```

## Key Concepts

### Max-Cut Problem
Given a graph, partition vertices into two sets to maximize edges between sets.

### QAOA Circuit
1. **Initial state**: Equal superposition of all partitions
2. **Problem layer**: Encodes graph structure (ZZ interactions)
3. **Mixer layer**: Explores new configurations (X rotations)
4. **Measure**: Sample good partitions

### Parameters
- **γ (gamma)**: Controls problem Hamiltonian strength
- **β (beta)**: Controls mixer strength
- **p**: Number of layers (more layers = better solution)

### Applications
- Network design
- Portfolio optimization
- Traffic routing
- VLSI circuit design
