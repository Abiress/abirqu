## Step-by-Step Code

```python
import math
import numpy as np
from abirqu import Circuit, H, X, CNOT, Rx, Ry, Rz, Measure
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Simple QNN architecture
# ============================================

print("Quantum Neural Network:")
print("=" * 50)

def build_qnn(num_qubits=2, depth=2, params=None):
    """Build quantum neural network circuit.
    
    Architecture: 
    - Encoding layer: Encode input
    - Variational layers: Parameterized gates
    - Measurement: Output prediction
    """
    circuit = Circuit(num_qubits, name="QNN")
    
    if params is None:
        params = [0.1] * (num_qubits * depth + num_qubits)
    
    param_idx = 0
    
    # Encoding layer
    for i in range(num_qubits):
        circuit.ry(i, params[param_idx])
        param_idx += 1
    
    # Variational layers
    for layer in range(depth):
        # Entanglement
        for i in range(num_qubits - 1):
            circuit.cnot(i, i + 1)
        
        # Rotations
        for i in range(num_qubits):
            circuit.ry(i, params[param_idx])
            param_idx += 1
    
    return circuit

# ============================================
# Part 2: Training loop
# ============================================

print("\nTraining QNN (simplified):")
print("-" * 50)

# XOR-like problem
X_train = [[0, 0], [0, 1], [1, 0], [1, 1]]
y_train = [0, 1, 1, 0]  # XOR

def predict(params, x):
    """Make prediction using QNN."""
    circuit = build_qnn(num_qubits=2, depth=2, params=params)
    
    # Encode input
    for i, val in enumerate(x):
        if val == 1:
            circuit.x(i)
    
    circuit.measure([0])
    result = QuantumRun(circuit, shots=100)
    
    # Return probability of |1⟩
    return result.probabilities.get('1', 0.5)

def loss(params, X, y):
    """Compute loss (MSE)."""
    total_loss = 0
    for x, target in zip(X, y):
        pred = predict(params, x)
        total_loss += (pred - target) ** 2
    return total_loss / len(X)

# Simple parameter search
print("Parameter search:")
best_loss = float('inf')
best_params = None

for _ in range(50):
    params = [np.random.uniform(-math.pi, math.pi) for _ in range(10)]
    current_loss = loss(params, X_train, y_train)
    
    if current_loss < best_loss:
        best_loss = current_loss
        best_params = params.copy()
        print(f"  Loss: {current_loss:.4f}")

# ============================================
# Part 3: Evaluation
# ============================================

print("\n\nEvaluation:")
print("=" * 50)

print("Predictions:")
for x, y_true in zip(X_train, y_train):
    y_pred = predict(best_params, x)
    print(f"  Input: {x} | Target: {y_true} | Prediction: {y_pred:.3f}")

print(f"\nFinal loss: {best_loss:.4f}")
```

## Key Concepts

### QNN Architecture
1. **Encoding**: Map classical input to quantum state
2. **Variational layers**: Parameterized rotations and entanglements
3. **Measurement**: Extract classical output

### Training
- Use classical optimizer to update quantum parameters
- Gradient computation via parameter shift rule
- Hybrid quantum-classical optimization

### Advantages
- Exponentially large Hilbert space
- Natural for quantum data
- Potential quantum advantage

### Applications
- **Classification**: Image recognition, NLP
- **Regression**: Time series prediction
- **Generative models**: Quantum GANs
- **Optimization**: Combinatorial problems
