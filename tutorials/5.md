## Step-by-Step Code

```python
from abirqu import Circuit
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: GHZ State (3 qubits)
# GHZ = (|000⟩ + |111⟩)/√2
# ============================================

print("GHZ State (3 qubits):")
print("=" * 40)

ghz = Circuit(3, name="GHZ State")

# Create superposition on first qubit
ghz.h(0)

# Propagate entanglement through CNOT chain
ghz.cnot(0, 1)
ghz.cnot(1, 2)

# Measure all
ghz.measure_all()

result = QuantumRun(ghz, shots=1000)
print(f"Counts: {result.counts}")
print(f"Expected: Only 000 and 111")

# ============================================
# Part 2: W State (3 qubits)
# W = (|001⟩ + |010⟩ + |100⟩)/√3
# ============================================

print("\n\nW State (3 qubits):")
print("=" * 40)

w = Circuit(3, name="W State")

# W state preparation circuit
w.x(0)           # Start with |100⟩
w.h(2)            # Put qubit 2 in superposition
w.cnot(2, 1)      # Entangle 2 and 1
w.cnot(2, 0)      # Entangle 2 and 0
w.x(2)            # Flip qubit 2

w.measure_all()

result = QuantumRun(w, shots=1000)
print(f"Counts: {result.counts}")
print(f"Expected: 001, 010, 100 with ~33% each")

# ============================================
# Part 2: 5-qubit GHZ State
# ============================================

print("\n\n5-qubit GHZ State:")
print("=" * 40)

ghz5 = Circuit(5, name="GHZ-5")
ghz5.h(0)
for i in range(4):
    ghz5.cnot(i, i + 1)
ghz5.measure_all()

result = QuantumRun(ghz5, shots=1000)
print(f"Counts: {result.counts}")
print(f"Expected: Only 00000 and 11111")
```

## Key Concepts

### GHZ vs W State
| Property | GHZ State | W State |
|----------|-----------|---------|
| Formula | (\|000⟩ + \|111⟩)/√2 | (\|001⟩ + \|010⟩ + \|100⟩)/√3 |
| Measurement | Only all-0s or all-1s | Exactly one 1 |
| Robustness | Fragile (lose 1 qubit = no entanglement) | Robust (lose 1 qubit = Bell pair) |
| Application | Quantum teleportation, secret sharing | Quantum networks, distributed computing |

### Entanglement Distribution
- GHZ: All qubits are maximally correlated
- W: Each qubit has equal probability of being |1⟩
