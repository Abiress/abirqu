## Step-by-Step Code

```python
import math
from abirqu import Circuit
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Classical Prisoner's Dilemma
# ============================================

print("Classical Prisoner's Dilemma:")
print("=" * 50)

# Payoff matrix: (Player1, Player2)
# Both Cooperate: (3, 3)
# Both Defect: (1, 1)
# One Cooperates, One Defects: (0, 5) or (5, 0)

print("Payoff Matrix (Player1, Player2):")
print("           | Cooperate | Defect")
print("-----------|-----------|--------")
print("Cooperate |  (3, 3)   | (0, 5)")
print("Defect    |  (5, 0)   | (1, 1)")
print("\nClassical Nash Equilibrium: Both Defect (1, 1)")

# ============================================
# Part 2: Quantum Prisoner's Dilemma
# ============================================

print("\n\nQuantum Prisoner's Dilemma:")
print("=" * 50)

def quantum_pd_circuit(strategy1, strategy2, entangle=True):
    """Build quantum PD circuit.
    
    Each player has 1 qubit.
    Strategy 0 = Cooperate, 1 = Defect.
    """
    circuit = Circuit(2, name="QPD")
    
    # Initialize both qubits
    if strategy1 == 1:  # Player 1 defects
        circuit.x(0)
    if strategy2 == 1:  # Player 2 defects
        circuit.x(1)
    
    # Entanglement (optional)
    if entangle:
        circuit.h(0)
        circuit.cnot(0, 1)
    
    # Payoff circuit
    # If both cooperate (|00⟩), give reward
    # If one defects, give temptation
    # If both defect, give punishment
    
    return circuit

# Test all strategy combinations
print("Testing all strategy combinations:")
for s1 in [0, 1]:  # Player 1
    for s2 in [0, 1]:  # Player 2
        circuit = quantum_pd_circuit(s1, s2, entangle=False)
        circuit.measure_all()
        
        result = QuantumRun(circuit, shots=100)
        
        # Determine outcome
        counts = result.counts
        if '00' in counts:
            outcome = "Both Cooperate"
            payoff = (3, 3)
        elif '01' in counts:
            outcome = "P1 Defects, P2 Cooperates"
            payoff = (5, 0)
        elif '10' in counts:
            outcome = "P1 Cooperates, P2 Defects"
            payoff = (0, 5)
        else:
            outcome = "Both Defect"
            payoff = (1, 1)
        
        print(f"  P1={'Defect' if s1 else 'Coop'}, "
              f"P2={'Defect' if s2 else 'Coop'}: "
              f"{outcome} → Payoff: {payoff}")

# ============================================
# Part 3: Quantum advantage
# ============================================

print("\n\nQuantum Advantage:")
print("=" * 50)

# With entanglement, quantum strategies can achieve better outcomes
print("With entanglement, players can achieve (3, 3) even with defection!")
print("This is the 'quantum' Nash equilibrium.")

# Demonstrate entangled strategy
circuit = Circuit(2, name="QPD_Entangled")

# Both players use Hadamard (quantum strategy)
circuit.h(0)
circuit.h(1)

# Entanglement
circuit.cnot(0, 1)

# Measure
circuit.measure_all()
result = QuantumRun(circuit, shots=1000)

print(f"Entangled strategy results: {result.counts}")
print("Superposition of outcomes → potential for cooperation!")
```

## Key Concepts

### Quantum Strategies
1. **Superposition**: Players can play multiple strategies simultaneously
2. **Entanglement**: Correlations between players' choices
3. **Interference**: Amplify desired outcomes

### Nash Equilibria
- **Classical**: Both Defect → (1, 1)
- **Quantum**: Entanglement enables (3, 3) equilibrium

### Applications
- **Economics**: Modeling market behavior
- **Negotiation**: Finding mutually beneficial solutions
- **Evolution**: Understanding cooperation in nature
- **Cryptography**: Designing secure protocols
