## Key Concepts

### Compilation Pipeline
1. **Decompose**: Break complex gates into basic gates
2. **Optimize**: Remove redundant gates
3. **Map**: Fit circuit to hardware connectivity
4. **Schedule**: Order gates for execution

### Gate Sets
- **Universal**: {H, T, CNOT}
- **Hardware-native**: Device-specific gates
- **Clifford+T**: Common fault-tolerant set

### Applications
- **Hardware execution**: Run circuits on real devices
- **Performance**: Reduce circuit depth
- **Cost**: Minimize gate count
- **Reliability**: Improve success rate
