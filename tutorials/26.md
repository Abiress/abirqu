## Step-by-Step Code

```python
import math
import numpy as np
from abirqu import Circuit, H, X, CNOT, Ry, Rz, Measure
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: User-Item matrix
# ============================================

print("Quantum Recommender System:")
print("=" * 50)

# Simple user-item rating matrix (5 users, 4 items)
# 0 = not rated
R = np.array([
    [5, 3, 0, 1],
    [4, 0, 0, 1],
    [1, 1, 0, 5],
    [1, 0, 0, 4],
    [0, 1, 5, 4],
])

print("User-Item Rating Matrix:")
print(f"Users × Items: {R.shape}")
print(R)

# ============================================
# Part 2: Quantum similarity
# ============================================

print("\n\nQuantum Similarity Computation:")
print("-" * 50)

def quantum_similarity_circuit(vec1, vec2):
    """Compute quantum similarity between two vectors."""
    circuit = Circuit(4, name="QSim")
    
    # Encode first vector
    circuit.ry(0, vec1[0] * math.pi / 5)
    circuit.ry(1, vec1[1] * math.pi / 5)
    
    # Encode second vector
    circuit.ry(2, vec2[0] * math.pi / 5)
    circuit.ry(3, vec2[1] * math.pi / 5)
    
    # Compute overlap
    circuit.h(0)
    circuit.cnot(0, 2)
    circuit.h(0)
    
    circuit.measure([0])
    return circuit

# Compute user-user similarities
print("User similarities (first 2 dimensions):")
for i in range(3):
    for j in range(i + 1, 3):
        circuit = quantum_similarity_circuit(R[i, :2], R[j, :2])
        result = QuantumRun(circuit, shots=100)
        similarity = result.probabilities.get('0', 0.5)
        print(f"  User {i} vs User {j}: {similarity:.3f}")

# ============================================
# Part 3: Recommendation
# ============================================

print("\n\nRecommendation Generation:")
print("=" * 50)

def recommend(user_id, R, k=2):
    """Generate recommendations for a user."""
    user_ratings = R[user_id]
    
    # Find unrated items
    unrated = np.where(user_ratings == 0)[0]
    
    # Simple prediction: average of similar users' ratings
    predictions = {}
    for item in unrated:
        # Find users who rated this item
        raters = np.where(R[:, item] != 0)[0]
        
        if len(raters) > 0:
            # Average rating
            predictions[item] = R[raters, item].mean()
        else:
            predictions[item] = 0
    
    # Sort by prediction score
    sorted_items = sorted(predictions.items(), key=lambda x: -x[1])
    return sorted_items[:k]

# Generate recommendations
for user in range(3):
    recs = recommend(user, R)
    print(f"  User {user} recommendations: {recs}")
```

## Key Concepts

### Quantum Recommender Algorithm
1. **Encode** user-item matrix in quantum states
2. **Compute similarities** using quantum overlap
3. **Predict** ratings using quantum matrix factorization
4. **Recommend** top-K items

### Quantum Advantages
- Exponential speedup in similarity computation
- Can handle large sparse matrices
- Natural for quantum data

### Applications
- **E-commerce**: Product recommendations
- **Streaming**: Content recommendations
- **Social networks**: Friend recommendations
- **Advertising**: Targeted ads
