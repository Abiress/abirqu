# Tutorial 2: Superposition and Measurement

## Objective
Understand quantum superposition by creating a single-qubit circuit and measuring probabilities.

## Prerequisites
- Tutorial 1 completed

## What You'll Learn
- Single-qubit superposition
- How measurement probabilities work
- Comparing results across different shot counts

## Step-by-Step Code

```python
from abirqu import Circuit, H, Measure
from abirqu.primitives import QuantumRun

# Create a 1-qubit circuit
circuit = Circuit(1, name="Superposition")

# Apply Hadamard gate — creates equal superposition
# |0⟩ → (|0⟩ + |1⟩)/√2
circuit.h(0)

# Measure the qubit
circuit.measure([0])

# Run with different shot counts to see statistics
for shots in [100, 1000, 10000]:
    result = QuantumRun(circuit, shots=shots)
    print(f"Shots: {shots:>5} | Counts: {result.counts}")
    print(f"           P(0)={result.probabilities.get('0', 0):.3f}  "
          f"P(1)={result.probabilities.get('1', 0):.3f}")
    print()
```

## Expected Output
```
Shots:   100 | Counts: {'0': 53, '1': 47}
           P(0)=0.530  P(1)=0.470

Shots:  1000 | Counts: {'0': 508, '1': 492}
           P(0)=0.508  P(1)=0.492

Shots: 10000 | Counts: {'0': 5012, '1': 4988}
           P(0)=0.501  P(1)=0.499
```

## Key Concepts

### Superposition
When you apply H to |0⟩, the qubit doesn't become 0 or 1. It exists in both states simultaneously. Only when you measure does it collapse to a definite value.

### Shot Count and Statistics
- **More shots = more accurate probabilities**
- With 100 shots, you might see 53/47 (far from ideal 50/50)
- With 10000 shots, you'll see approximately 5000/5000
- This is quantum randomness — not pseudorandom like classical computers

### The Born Rule
The probability of measuring |0⟩ is |α|² and |1⟩ is |β|², where α and β are the amplitudes. For H|0⟩, both are 1/√2, so both probabilities are 1/2.
