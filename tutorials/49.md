class EntanglementManager:
    """Manage entanglement across network."""
    
    def __init__(self):
        self.pools = {}
    
    def store(self, node, pair_id):
        """Store entanglement pair."""
        if node not in self.pools:
            self.pools[node] = []
        self.pools[node].append(pair_id)
    
    def retrieve(self, node):
        """Retrieve entanglement pair."""
        if node in self.pools and self.pools[node]:
            return self.pools[node].pop()
        return None
    
    def get_stats(self):
        """Get entanglement statistics."""
        return {node: len(pairs) for node, pairs in self.pools.items()}

manager = EntanglementManager()

# Generate entanglement
for i in range(5):
    manager.store(0, f"pair_{i}")
    manager.store(1, f"pair_{i}")

print(f"Entanglement pools: {manager.get_stats()}")

# Use entanglement
pair = manager.retrieve(0)
print(f"Retrieved pair from node 0: {pair}")

## Key Concepts

### Quantum Internet Layers
1. **Physical**: Hardware (detectors, sources, channels)
2. **Link**: Entanglement generation between neighbors
3. **Network**: Entanglement routing across network
4. **Transport**: End-to-end entanglement management
5. **Application**: User-facing quantum services

### Entanglement Management
- **Generation**: Create entangled pairs
- **Storage**: Maintain entanglement (challenging!)
- **Purification**: Improve fidelity
- **Swapping**: Extend distance

### Applications
- **QKD**: Unconditionally secure communication
- **Distributed QC**: Access remote quantum computers
- **Quantum sensing**: Global sensor networks
- **Quantum money**: Unforgeable currency
