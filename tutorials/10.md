## Step-by-Step Code

```python
import math
from abirqu import Circuit, H, X, Y, Z
from abirqu.primitives import QuantumRun
from abirqu.library import vqe_hardware_efficient, real_amplitudes

# ============================================
# Part 1: Simple VQE for H2 molecule
# ============================================

print("VQE for H2 Molecule (Simplified):")
print("=" * 50)

# H2 Hamiltonian in Pauli basis (simplified)
# H = c0*I + c1*Z0 + c2*Z1 + c3*Z0Z1 + c4*X0X1
coefficients = {
    'I': -0.4804,
    'Z0': 0.3435,
    'Z1': 0.3435,
    'Z0Z1': -0.4347,
    'X0X1': 0.0910,
}

def compute_energy(params, shots=1000):
    """Compute expectation value of Hamiltonian."""
    # Create ansatz circuit
    circuit = Circuit(2)
    
    # Parameterized ansatz
    circuit.ry(0, params[0])
    circuit.ry(1, params[1])
    circuit.cnot(0, 1)
    circuit.ry(0, params[2])
    
    # Measure each Pauli term
    energy = coefficients['I']  # Identity contribution
    
    # Z0 term
    circuit.measure_all()
    result = QuantumRun(circuit, shots=shots)
    p0 = result.probabilities.get('0', 0)
    p1 = result.probabilities.get('1', 0)
    energy += coefficients['Z0'] * (p0 - p1)
    
    return energy

# ============================================
# Part 2: Optimization loop
# ============================================

print("\nOptimization:")
print("-" * 50)

# Simple grid search for demonstration
best_energy = float('inf')
best_params = None

for t0 in [i * 0.1 for i in range(10)]:
    for t1 in [i * 0.1 for i in range(10)]:
        for t2 in [i * 0.1 for i in range(10)]:
            params = [t0, t1, t2]
            energy = compute_energy(params)
            if energy < best_energy:
                best_energy = energy
                best_params = params

print(f"Best parameters: {[f'{p:.1f}' for p in best_params]}")
print(f"Ground state energy: {best_energy:.6f} Ha")
print(f"Exact H2 energy: -1.1373 Ha (approximate)")
print(f"Error: {abs(best_energy - (-1.1373)):.6f} Ha")

# ============================================
# Part 3: Using built-in VQE
# ============================================

print("\n\nUsing built-in VQE ansatz:")
print("=" * 50)

ansatz = vqe_hardware_efficient(num_qubits=2, depth=2)
from abirqu.visualization import draw
print(draw(ansatz, output="text"))
print(f"Ansatz parameters: {sum(1 for g in ansatz.gates if g.params)}")
```

## Key Concepts

### VQE Algorithm
1. **Prepare** trial state |ψ(θ)⟩ using parameterized circuit
2. **Measure** expectation value ⟨ψ(θ)|H|ψ(θ)⟩
3. **Optimize** parameters θ to minimize energy
4. **Repeat** until convergence

### Why Variational?
- NISQ-friendly: shallow circuits, few measurements
- Classical optimizer handles the hard part
- Robust to noise (noise doesn't prevent convergence)

### Ansatz Choices
- **Hardware-efficient**: Uses native gates of the device
- **UCCSD**: Chemically inspired, more accurate but deeper
- **ADAPT-VQE**: Grows circuit dynamically

### Energy Estimation
For Hamiltonian H = Σᵢ cᵢPᵢ where Pᵢ are Pauli operators:
- Measure each Pᵢ separately
- Combine: E = Σᵢ cᵢ⟨Pᵢ⟩
