## Step-by-Step Code

```python
import math
import numpy as np
from abirqu import Circuit, H, X, Y, Z
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Classical solution for comparison
# ============================================

print("HHL Algorithm Demo:")
print("=" * 50)

# Simple 2x2 system: Ax = b
A = np.array([[2, 1], [1, 3]])
b = np.array([1, 2])

# Classical solution
x_classical = np.linalg.solve(A, b)
print(f"Matrix A:\n{A}")
print(f"Vector b: {b}")
print(f"Classical solution x: {x_classical}")

# ============================================
# Part 2: Quantum solution (conceptual)
# ============================================

print("\n\nQuantum Solution (conceptual HHL circuit):")
print("=" * 50)

# HHL algorithm steps:
# 1. Encode b in quantum state |b⟩
# 2. Apply phase estimation to get eigenvalues
# 3. Rotate ancilla qubit by 1/λ
# 4. Uncompute and measure

# For demonstration, we'll show the circuit structure
def build_hhl_circuit(num_qubits=4):
    """Build conceptual HHL circuit."""
    circuit = Circuit(num_qubits, name="HHL")
    
    # Encode input vector (simplified)
    circuit.h(0)
    
    # Phase estimation (conceptual)
    for i in range(2):
        circuit.h(i + 1)
    
    # Controlled rotations (1/λ)
    circuit.ry(3, math.pi / 4)
    
    # Uncompute
    for i in range(2):
        circuit.h(i + 1)
    
    return circuit

hhl = build_hhl_circuit()
from abirqu.visualization import draw
print(draw(hhl, output="text"))

# ============================================
# Part 3: Using AbirQu's HHL solver
# ============================================

print("\n\nUsing AbirQu's HHL Solver:")
print("=" * 50)

from abirqu.space import HHLSolver

solver = HHLSolver(n_qubits=4)
# The solver handles all the quantum circuit construction
print("HHL Solver available in abirqu.space")
print("Usage: solver.solve(A, b)")
print(f"For A={A.tolist()}, b={b.tolist()}")
print(f"Expected: x ≈ {x_classical}")
```

## Key Concepts

### HHL Algorithm
1. **Encode** b as quantum state |b⟩
2. **Phase estimation** to get eigenvalues λᵢ of A
3. **Rotate** ancilla by 1/λᵢ (controlled rotation)
4. **Uncompute** and measure to get |x⟩

### Complexity
- Classical: O(N³) for N×N matrix
- HHL: O(log(N)³) — exponential speedup!

### Limitations
- Requires sparse matrix
- Solution is in quantum state (need tomography to extract)
- Not always faster than classical

### Applications
- Quantum chemistry (solving Schrödinger equation)
- Machine learning (kernel methods)
- Optimization (interior point methods)
- Fluid dynamics (CFD)
