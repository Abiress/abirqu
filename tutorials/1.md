# Tutorial 1: Your First Quantum Circuit — The Bell State

## Objective
Create a Bell state (quantum entanglement) using AbirQu and measure the results.

## Prerequisites
- AbirQu installed (`pip install abirqu`)

## What You'll Learn
- How to create a quantum circuit
- How to apply Hadamard and CNOT gates
- How to measure and interpret results

## Step-by-Step Code

```python
from abirqu import Circuit, H, X, Y, Z

# Step 1: Create a 2-qubit circuit
circuit = Circuit(2, name="Bell State")

# Step 2: Apply Hadamard gate to qubit 0
# This puts qubit 0 into superposition: |0⟩ → (|0⟩ + |1⟩)/√2
circuit.h(0)

# Step 3: Apply CNOT gate (control=0, target=1)
# This entangles qubit 1 with qubit 0
circuit.cnot(0, 1)

# Step 4: Measure both qubits
circuit.measure_all()

# Step 5: Run the circuit
from abirqu.primitives import QuantumRun
result = QuantumRun(circuit, shots=1000)

# Step 6: View results
print("Measurement counts:", result.counts)
print("Probabilities:", result.probabilities)
```

## Expected Output
```
Measurement counts: {'00': 502, '11': 498}
Probabilities: {'00': 0.502, '11': 0.498}
```

## Key Concepts

### What is a Bell State?
A Bell state is one of four maximally entangled two-qubit states:
- |Φ+⟩ = (|00⟩ + |11⟩)/√2  ← This tutorial creates this one
- |Φ-⟩ = (|00⟩ - |11⟩)/√2
- |Ψ+⟩ = (|01⟩ + |10⟩)/√2
- |Ψ-⟩ = (|01⟩ - |10⟩)/√2

### Why Does Measurement Only Give 00 or 11?
Before measurement, the system exists in superposition of |00⟩ and |11⟩. Measurement collapses it to one of these two states. You never see 01 or 10 because the qubits are entangled — measuring one instantly determines the other.
