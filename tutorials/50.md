## Key Concepts

### Hardware Platforms
1. **Superconducting**: Fast gates, scalable, needs cooling
2. **Trapped Ion**: High fidelity, long coherence, slow gates
3. **Photonic**: Room temperature, hard to make interactions
4. **Topological**: Inherently protected, still experimental

### Noise Sources
- **Decoherence**: T1 (energy loss), T2 (phase loss)
- **Gate errors**: Imperfect control pulses
- **Readout errors**: Measurement mistakes
- **Crosstalk**: Unwanted interactions

### Applications
- **NISQ algorithms**: VQE, QAOA on current devices
- **Error correction**: Fault-tolerant computing
- **Quantum advantage**: Solving classically hard problems
