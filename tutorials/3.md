# Tutorial 3: Multi-Qubit Circuits and Gate Combinations

## Objective
Build circuits with multiple gates and understand how gates compose.

## Prerequisites
- Tutorials 1-2 completed

## What You'll Learn
- Applying multiple gates in sequence
- Gate composition and circuit depth
- Creating different quantum states

## Step-by-Step Code

```python
from abirqu import Circuit, H, X, Y, Z
from abirqu.primitives import QuantumRun
from abirqu.visualization import draw

# Create a 3-qubit circuit
circuit = Circuit(3, name="Multi-Qubit Demo")

# Apply different single-qubit gates
circuit.h(0)       # Qubit 0: superposition
circuit.x(1)       # Qubit 1: bit flip (|0⟩ → |1⟩)
circuit.y(2)       # Qubit 2: Y rotation

# Apply entangling gates
circuit.cnot(0, 1)  # Entangle qubits 0 and 1
circuit.cnot(1, 2)  # Entangle qubits 1 and 2

# Measure all qubits
circuit.measure_all()

# Run
result = QuantumRun(circuit, shots=1000)
print("Counts:", result.counts)

# Visualize the circuit
print("\nCircuit diagram:")
print(draw(circuit, output="text"))
```

## Expected Output
```
Counts: {'010': 252, '011': 248, '110': 253, '111': 247}

Circuit diagram:
q0: ──H──●──────M──
          │
q1: ──X──⊕──●──M──
             │
q2: ──Y──────⊕──M──
```

## Key Concepts

### Gate Order Matters
Gates are applied left to right. `H(0)` then `CNOT(0,1)` is different from `CNOT(0,1)` then `H(0)`.

### Circuit Depth
The depth is the longest path through the circuit. This circuit has depth 3 (H → CNOT → CNOT).

### The State After This Circuit
- Qubit 0: H|0⟩ = (|0⟩ + |1⟩)/√2
- Qubit 1: X|0⟩ = |1⟩, then CNOT flips it based on qubit 0
- Qubit 2: Y|0⟩ = i|1⟩, then CNOT flips it based on qubit 1
