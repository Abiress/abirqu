## Step-by-Step Code

```python
import math
from abirqu import Circuit
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Quantum computing models
# ============================================

print("Expert Quantum Computing (Part 2):")
print("=" * 50)

models = {
    'Circuit Model': {
        'description': 'Sequential gate operations',
        'universality': 'Universal',
        'advantage': 'Most intuitive',
    },
    'Measurement-Based': {
        'description': 'Single-qubit measurements on cluster state',
        'universality': 'Universal',
        'advantage': 'Natural for some algorithms',
    },
    'Topological': {
        'description': 'Anyonic braiding',
        'universality': 'Universal',
        'advantage': 'Error-resistant',
    },
    'Adiabatic': {
        'description': 'Slow Hamiltonian evolution',
        'universality': 'Universal',
        'advantage': 'Natural for optimization',
    },
    'Quantum Walk': {
        'description': 'Discrete-time quantum walk',
        'universality': 'Universal',
        'advantage': 'Graph algorithms',
    },
}

for model, properties in models.items():
    print(f"\n{model}:")
    for prop, value in properties.items():
        print(f"  {prop}: {value}")

# ============================================
# Part 2: Complexity theory
# ============================================

print("\n\nQuantum Complexity Classes:")
print("-" * 50)

classes = [
    ("BQP", "Bounded-error Quantum Polynomial", "P ⊆ BQP ⊆ PSPACE"),
    ("QMA", "Quantum Merlin-Arthur", "Quantum version of NP"),
    ("QIP", "Quantum Interactive Proofs", "PSPACE-complete"),
    ("QCMA", "Quantum Classical Merlin-Arthur", "QMA with classical proofs"),
]

for name, full_name, relationship in classes:
    print(f"  {name}: {full_name}")
    print(f"    Relationship: {relationship}")

# ============================================
# Part 3: Quantum algorithms
# ============================================

print("\n\nQuantum Algorithm Families:")
print("-" * 50)

families = [
    ("Amplitude Amplification", "Grover, QAOA", "Quadratic speedup"),
    ("Phase Estimation", "Shor, HHL", "Exponential speedup"),
    ("Quantum Walk", "Element distinctness", "Polynomial speedup"),
    ("Variational", "VQE, QAOA", "NISQ-friendly"),
    ("Topological", "Surface codes", "Error correction"),
]

for family, examples, advantage in families:
    print(f"  {family:30s}: {examples:20s} ({advantage})")

# Final demonstration
print("\n\nFinal Demonstration:")
print("-" * 50)

circuit = Circuit(3, name="Final")
circuit.h(0)
circuit.cnot(0, 1)
circuit.cnot(1, 2)
circuit.measure_all()

result = QuantumRun(circuit, shots=1000)
print(f"GHZ state distribution: {result.counts}")
```

## Key Concepts

### Quantum Computing Models
- **Circuit**: Most common, gate-based
- **Measurement-based**: Cluster states
- **Topological**: Error-resistant
- **Adiabatic**: Natural for optimization

### Complexity Classes
- **BQP**: Efficiently solvable by quantum computers
- **QMA**: Quantum analog of NP
- **PSPACE**: Problems solvable with polynomial space

### Algorithm Families
- **Amplitude amplification**: Grover-type speedups
- **Phase estimation**: Shor-type speedups
- **Variational**: NISQ-friendly algorithms
