## Step-by-Step Code

```python
import math
import numpy as np
from abirqu import Circuit
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Quantum Otto cycle
# ============================================

print("Quantum Thermodynamics:")
print("=" * 50)

def quantum_otto_step(circuit, step_type, params):
    """Apply Otto cycle step."""
    if step_type == 'isochoric_hot':
        # Hot bath coupling
        circuit.ry(0, params['temperature'])
    elif step_type == 'expansion':
        # Work extraction
        circuit.rz(0, params['frequency'])
    elif step_type == 'isochoric_cold':
        # Cold bath coupling
        circuit.ry(0, params['temperature'] * 0.3)
    elif step_type == 'compression':
        # Work input
        circuit.rz(0, -params['frequency'])

print("Quantum Otto Cycle:")
print("-" * 50)

# Simulate one cycle
circuit = Circuit(1, name="Otto")

params = {
    'temperature': math.pi / 4,
    'frequency': math.pi / 6,
}

# Initial state
circuit.h(0)

# Cycle steps
quantum_otto_step(circuit, 'isochoric_hot', params)
quantum_otto_step(circuit, 'expansion', params)
quantum_otto_step(circuit, 'isochoric_cold', params)
quantum_otto_step(circuit, 'compression', params)

circuit.measure_all()
result = QuantumRun(circuit, shots=1000)

print(f"Final state distribution: {result.counts}")

# ============================================
# Part 2: Efficiency calculation
# ============================================

print("\n\nEfficiency Analysis:")
print("-" * 50)

# Otto cycle efficiency: η = 1 - T_c/T_h
T_h = 1.0  # Hot temperature
T_c = 0.3  # Cold temperature

eta_otto = 1 - T_c / T_h
eta_carnot = 1 - T_c / T_h

print(f"Hot temperature: {T_h}")
print(f"Cold temperature: {T_c}")
print(f"Otto efficiency: {eta_otto:.3f}")
print(f"Carnot efficiency: {eta_carnot:.3f}")

# ============================================
# Part 3: Quantum work extraction
# ============================================

print("\n\nQuantum Work Extraction:")
print("-" * 50)

def compute_work(circuit, shots=1000):
    """Estimate work from quantum measurements."""
    measure_circuit = circuit.copy()
    measure_circuit.measure_all()
    
    result = QuantumRun(measure_circuit, shots=shots)
    
    # Work estimate (simplified)
    work = 0
    for bitstring, count in result.counts.items():
        if bitstring == '0':
            work += count / shots
        else:
            work -= count / shots
    
    return work

work = compute_work(circuit)
print(f"Extracted work: {work:.3f}")
print(f"Work is positive: {work > 0}")
```

## Key Concepts

### Quantum Otto Cycle
1. **Isochoric heating**: System absorbs heat from hot bath
2. **Expansion**: System does work
3. **Isochoric cooling**: System releases heat to cold bath
4. **Compression**: Work is done on system

### Quantum Thermodynamic Relations
- **Efficiency**: η = W/Q_h ≤ 1 - T_c/T_h (Carnot limit)
- **Work**: W = ΔE - Q
- **Entropy**: S = -Tr(ρ log ρ)

### Applications
- **Quantum engines**: Miniature power sources
- **Quantum refrigerators**: Cooling quantum devices
- **Energy harvesting**: Quantum solar cells
- **Quantum batteries**: Energy storage
