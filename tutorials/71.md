## Step-by-Step Code

```python
import math
from abirqu import Circuit
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Basic quantum RNG
# ============================================

print("Quantum Random Number Generation:")
print("=" * 50)

def quantum_random_bit():
    """Generate single random bit using quantum circuit."""
    circuit = Circuit(1, name="QRNG")
    circuit.h(0)
    circuit.measure_all()
    
    result = QuantumRun(circuit, shots=1)
    return int(list(result.counts.keys())[0])

# Generate random numbers
print("Generating random bits:")
bits = [quantum_random_bit() for _ in range(20)]
print(f"  20 random bits: {''.join(map(str, bits))}")

# ============================================
# Part 2: Random number in range
# ============================================

print("\n\nRandom Numbers in Range:")
print("-" * 50)

def quantum_random_int(min_val, max_val, num_bits=8):
    """Generate random integer in range."""
    # Generate enough bits
    circuit = Circuit(num_bits, name="QRNG_Int")
    for i in range(num_bits):
        circuit.h(i)
    circuit.measure_all()
    
    result = QuantumRun(circuit, shots=1)
    bitstring = list(result.counts.keys())[0]
    
    # Convert to integer
    value = int(bitstring, 2)
    
    # Scale to range
    range_size = max_val - min_val + 1
    scaled = min_val + (value % range_size)
    
    return scaled

print("Random numbers 1-100:")
random_numbers = [quantum_random_int(1, 100) for _ in range(10)]
print(f"  {random_numbers}")

# ============================================
# Part 3: Randomness testing
# ============================================

print("\n\nRandomness Testing:")
print("-" * 50)

def test_randomness(bits):
    """Simple randomness tests."""
    n = len(bits)
    ones = sum(bits)
    
    # Frequency test
    freq_ratio = ones / n
    
    # Runs test
    runs = 1
    for i in range(1, n):
        if bits[i] != bits[i-1]:
            runs += 1
    
    return {
        'frequency': freq_ratio,
        'runs': runs,
        'balanced': 0.4 < freq_ratio < 0.6,
    }

# Generate and test
bits = [quantum_random_bit() for _ in range(1000)]
tests = test_randomness(bits)

print(f"  Frequency: {tests['frequency']:.3f} (should be ~0.5)")
print(f"  Runs: {tests['runs']} (should be ~500)")
print(f"  Balanced: {tests['balanced']}")

# ============================================
# Part 4: Applications
# ============================================

print("\n\nApplications:")
print("-" * 50)

applications = [
    "Cryptography: Key generation",
    "Monte Carlo simulation",
    "Lottery and gambling",
    "Scientific research",
    "Statistical sampling",
]

for app in applications:
    print(f"  - {app}")
```

## Key Concepts

### Quantum Randomness
- **True randomness**: Based on quantum mechanics
- **Certifiable**: Can prove randomness
- **Unpredictable**: No hidden variables

### How It Works
1. Prepare qubit in |0⟩
2. Apply Hadamard: (|0⟩ + |1⟩)/√2
3. Measure: random 0 or 1
4. Born's rule: P(0) = P(1) = 0.5

### Advantages Over Classical
- **Provably random**: Not pseudo-random
- **Unpredictable**: Even with unlimited computing
- **Certifiable**: Can verify randomness

### Applications
- **Cryptography**: Key generation, nonces
- **Simulation**: Monte Carlo methods
- **Gaming**: Fair randomness
- **Research**: Statistical sampling
