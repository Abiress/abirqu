## Step-by-Step Code

```python
import math
from abirqu import Circuit, H, X, CNOT, Rz, Measure
from abirqu.primitives import QuantumRun

# ============================================
# Part 1: Simple QRAM concept
# ============================================

print("Quantum Random Access Memory:")
print("=" * 50)

def build_qram_circuit(address, data, num_address_qubits=2):
    """Build QRAM circuit.
    
    Address register → Data register
    """
    num_data_qubits = len(data[0]) if data else 2
    total_qubits = num_address_qubits + num_data_qubits
    
    circuit = Circuit(total_qubits, name="QRAM")
    
    # Encode address
    for i in range(num_address_qubits):
        if address[i] == 1:
            circuit.x(i)
    
    # Memory lookup (simplified)
    # In real QRAM, this would be a complex oracle
    for i in range(num_data_qubits):
        circuit.cnot(0, num_address_qubits + i)
    
    return circuit

# Simple memory: address → data
memory = {
    '00': [0, 1],
    '01': [1, 0],
    '10': [1, 1],
    '11': [0, 0],
}

print("Memory contents:")
for addr, data in memory.items():
    print(f"  |{addr}⟩ → |{data}⟩")

# ============================================
# Part 2: QRAM access
# ============================================

print("\n\nQRAM Access:")
print("-" * 50)

for addr in ['00', '01', '10', '11']:
    address = [int(b) for b in addr]
    circuit = build_qram_circuit(address, [memory[addr]])
    circuit.measure([2, 3])  # Measure data register
    
    result = QuantumRun(circuit, shots=100)
    retrieved = max(result.counts, key=result.counts.get)
    print(f"  Address |{addr}⟩ → Retrieved |{retrieved}⟩")

# ============================================
# Part 3: Superposition query
# ============================================

print("\n\nSuperposition Query:")
print("-" * 50)

# Query multiple addresses in superposition
circuit = Circuit(4, name="QRAM_Superposition")

# Put address in superposition
circuit.h(0)
circuit.h(1)

# QRAM oracle (simplified)
circuit.cnot(0, 2)
circuit.cnot(1, 3)

circuit.measure([2, 3])

result = QuantumRun(circuit, shots=1000)
print("Superposition query results:")
print(f"  Counts: {result.counts}")
print("  All data retrieved in superposition!")
```

## Key Concepts

### QRAM Architecture
1. **Address register**: Specifies which memory location
2. **Data register**: Contains the stored data
3. **Oracle**: Maps address to data

### QRAM Advantages
- **Superposition query**: Access all addresses simultaneously
- **Exponential speedup**: For certain algorithms
- **Natural for quantum algorithms**: Grover's, HHL, etc.

### Applications
- **Database search**: Query entire database at once
- **Machine learning**: Access training data
- **Optimization**: Load problem data
- **Simulation**: Access Hamiltonian terms
