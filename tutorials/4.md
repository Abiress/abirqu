## Step-by-Step Code

```python
import math
from abirqu import Circuit, H, Rx, Ry, Rz, Measure
from abirqu.primitives import QuantumRun

# Experiment with different rotation angles
angles = [0, math.pi/4, math.pi/2, 3*math.pi/4, math.pi]

print("Ry Rotation Experiment:")
print("=" * 50)

for angle in angles:
    circuit = Circuit(1, name=f"Ry({angle:.2f})")
    circuit.ry(0, angle)
    circuit.measure([0])
    
    result = QuantumRun(circuit, shots=1000)
    p0 = result.probabilities.get('0', 0)
    p1 = result.probabilities.get('1', 0)
    
    print(f"θ = {angle:.4f} rad ({math.degrees(angle):.1f}°) | "
          f"P(0) = {p0:.3f} | P(1) = {p1:.3f}")

# Create a specific state: 75% |0⟩, 25% |1⟩
print("\n\nCreating custom state (75% |0⟩, 25% |1⟩):")
circuit = Circuit(1)
# Ry(θ) where cos²(θ/2) = 0.75, so θ = 2*arccos(√0.75)
theta = 2 * math.acos(math.sqrt(0.75))
circuit.ry(0, theta)
circuit.measure([0])

result = QuantumRun(circuit, shots=1000)
print(f"Target: P(0)=0.750, P(1)=0.250")
print(f"Actual: P(0)={result.probabilities.get('0', 0):.3f}, "
      f"P(1)={result.probabilities.get('1', 0):.3f}")
```

## Key Concepts

### Rotation Gate Formulas
- **Ry(θ)**: Rotates around Y-axis. P(0) = cos²(θ/2), P(1) = sin²(θ/2)
- **Rx(θ)**: Rotates around X-axis. Creates complex amplitudes
- **Rz(θ)**: Rotates around Z-axis. Only changes phase, no measurement difference

### Bloch Sphere
Every single-qubit state can be represented as a point on the Bloch sphere:
- θ (polar angle) controls |0⟩ vs |1⟩ probability
- φ (azimuthal angle) controls phase

### Practical Use
Parameterized gates are the building blocks of:
- Variational quantum algorithms (VQE, QAOA)
- Quantum machine learning
- Quantum control and optimization
