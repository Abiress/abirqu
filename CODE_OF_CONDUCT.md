# AbirQu Code of Conduct
Copyright 2026 Abir Maheshwari

## 1. Pledge
Abir Maheshwari pledges that participation in AbirQu is a harassment-free experience for all contributors and users.

## 2. Standards
- Be respectful of differing viewpoints and experiences
- Accept constructive criticism gracefully
- Focus on actions that benefit the project
- Avoid personal attacks, trolling, or discriminatory language

## 3. Enforcement
Abir Maheshwari is solely responsible for enforcing this Code of Conduct. Report violations via GitHub issues or direct message.
